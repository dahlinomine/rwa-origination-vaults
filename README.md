## Overview

**rwa-origination-vaults** is an open-source smart-contract for Real World Asset tokenization and on-chain finance.

Solidity contracts for on-chain credit origination using Morpho Vaults as the liquidity back-end.

Topics: `defi` `lending` `tokenization`

## Features

- 🚀 Production-ready smart-contract targeting rwa use cases
- 🔧 Easy to integrate into existing projects
- 📦 Zero-configuration defaults with full customization support
- 🧪 Comprehensive test coverage
- 📖 Well-documented API with examples
